== Handling Images

=== Update Ruby

.Update ruby base image
[source,sh]
----
podman pull docker.io/library/ruby:3.2.6-slim
----

=== Build Image

.Build
[source,sh]
----
podman build -t cocard .
----

.Inspect image
[source,sh]
----
podman run --rm -it cocard /bin/bash
----

=== Tag Image

.Tag image
[source,sh]
----
podman tag cocard ghcr.io/swobspace/cocard:0.6.0
podman tag cocard ghcr.io/swobspace/cocard:latest
----

=== Push Image

.Push image
[source,sh]
----
export CR_PAT=YOUR_TOKEN
echo $CR_PAT | podman login ghcr.io -u swobspace --password-stdin
podman push ghcr.io/swobspace/cocard:0.6.0
podman push ghcr.io/swobspace/cocard:latest
----
